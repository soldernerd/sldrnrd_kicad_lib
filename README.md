# sldrnrd KiCad Library

Custom KiCad component library by soldernerd.

## Contents

| Folder | Description |
|--------|-------------|
| `symbols/` | Schematic symbol libraries, organized by category |
| `sldrnrd.pretty/` | PCB footprint library |
| `3dmodels/` | 3D models for footprints (.step / .wrl) |
| `datasheets/` | Component datasheets |

## Usage

Add the library to KiCad via **Preferences → Manage Symbol Libraries** (for symbols) and **Preferences → Manage Footprint Libraries** (for footprints).
